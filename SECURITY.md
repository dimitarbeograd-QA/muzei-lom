# Security Policy

## Докладване на уязвимост
Ако откриеш уязвимост в този сайт, моля не отваряй публичен Issue.
Свържи се директно: **dimitar_beograd@abv.bg**

## Известни съображения
- Admin панелът (`admin-nav-btn`, `openAdminPanel`) трябва да изисква реална
  автентикация. Ако управлението на съдържание е само client-side, всеки
  посетител на сайта потенциално може да достигне до admin функциите през
  DevTools — препоръчително е admin действия да минават през защитен backend.
- Ако бъде добавена форма за контакт/коментари, задължително добави
  input validation и anti-spam защита (напр. honeypot поле).
