# Contributing

## Как да допринесеш
1. Направи промените директно в `index.html`.
2. Тествай ръчно по чеклиста в [`qa-docs/TEST_CASES.md`](qa-docs/TEST_CASES.md) — особено навигация, timeline и 3D изгледа.
3. При добавяне на нов отдел/експозиция — провери, че е консистентен по стил с останалите карти.
4. Тествай и в двете теми (light/dark) преди merge.

## Docs
При добавяне на нова функционалност, обнови съответно:
- [`README.md`](README.md)
- [`qa-docs/TEST_CASES.md`](qa-docs/TEST_CASES.md)
- [`CHANGELOG.md`](CHANGELOG.md)
