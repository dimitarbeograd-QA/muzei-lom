# Test Plan — Исторически музей Лом

## 1. Обхват
Уебсайт на Исторически музей — Лом с раздели: За нас, Алмус (Almus), Хронология
(timeline с плъзгач), 3D Алмус изглед, Отдели, Експозиции, Събития (условно
видими), Новини, Посещение. Включва тъмна/светла тема, text-to-speech (TTS) с
избор на език, AI чат гид "Алмус" (Anthropic Claude API), admin панел (с AI
генератор на съдържание), бутони за споделяне на секции, PWA install банер,
интерактивна римска карта, отделна музейна карта и 6 сайт-широки дизайн
шаблона (admin таб "Дизайн"), всеки с dark/light вариант.

## 2. Цели на тестването
- Навигацията (nav линкове, mobile меню) да скролва коректно до всяка секция.
- Timeline плъзгачът да показва правилното историческо съдържание за всеки период.
- 3D Алмус изгледът (front/top/side/360°, fullscreen, highlight) да работи без визуални грешки.
- TTS четенето на съдържание да работи на избрания език.
- Тъмна/светла тема да превключва коректно всички стилове (включително карти, timeline, снимки).
- Admin панелът да е защитен и недостъпен за обикновени посетители.
- AI чатът "Алмус" да показва ясно съобщение, когато не е конфигуриран API ключ,
  вместо да замръзва или да хвърля JS грешка.
- Admin AI генераторът на съдържание да изисква валиден API ключ преди заявка.
- Избраният дизайн шаблон да важи сайт-широко (не per-visitor) и да работи
  коректно в комбинация с light/dark темата.

## 3. Тестова среда
| Компонент | Детайли |
|---|---|
| Браузъри | Chrome, Firefox, Edge, Safari |
| Устройства | Desktop, Tablet, Mobile |
| Теми | Light theme, Dark theme (default) |
| Езици (TTS) | Български + допълнителни, ако са налични |

## 4. Видове тестове
1. **Функционално** — навигация, timeline, 3D изглед, TTS, тема, споделяне.
2. **UI/UX** — responsive поведение, mobile меню, четимост на текст в двете теми.
3. **Достъпност (a11y)** — TTS, контраст на цветовете, keyboard navigation.
4. **Roles/Security** — admin панел скрит/защитен за неоторизирани потребители.
5. **Regression** — при добавяне на нови отдели/експозиции/събития.

## 5. Критерии за приемане
- Всички nav линкове водят до правилната секция без счупен scroll.
- Timeline слайдерът показва коректно съдържание за всяка от 6-те позиции (0-5).
- 3D Алмус изгледът се зарежда без забавяне > 3с и работи във fullscreen.
- Тема (light/dark) се запазва между презареждания (ако е предвидено с localStorage).
- Admin панел изисква автентикация.
- AI чатът никога не прави заявка с празен/невалиден ключ мълчаливо — винаги
  информира потребителя.
